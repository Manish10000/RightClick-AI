# Extension Icons

Please add the following icon files to this directory:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

You can:
1. Create simple icons using any image editor
2. Use online icon generators
3. Convert any PNG images to the required sizes

The extension will not load properly without these icon files.
