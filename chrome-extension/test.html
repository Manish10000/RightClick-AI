🧪 AI Keyboard Extension Test Page

📋 How to Test:

  1. Make sure the AI Keyboard extension is installed and your RAG server is running
  2. Select any text below by clicking and dragging
  3. Right-click to open the context menu
  4. Choose from "AI Keyboard Assistant" options
  5. Wait for the AI response to appear in a floating window

🚨 Troubleshooting:

  • Context menu not appearing: Refresh this page and try again
  • No response from AI: Check if your RAG server is running on the correct URL
  • Authentication errors: Login through the extension popup first
  • RAG replies not working: Upload some documents first through the extension

⚙️ Extension Settings:

Click the extension icon in your Chrome toolbar to:

  • Configure server URL (default: http://localhost:8000)
  • Login/register for user accounts
  • Upload PDF documents for RAG functionality
  • View and manage your uploaded documents
